## 🚀 Getting Started

Wanna run this locally? Follow these steps:

1. **Clone the Repo**
   ```bash
   git clone https://github.com/noorulameen17/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the Project**
   ```bash
   npm start
   ```

4. **Open in Browser**
   ```bash
   http://localhost:3000
   ```
